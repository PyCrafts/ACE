    Analysis Overview¶

    ACE analyzes Observables which generates Analysis output which can in turn contain more Observables.

    Observable¶

    An Observable is an observation made during analysis. It has

    • a type.
    • a value.
    • an optional time at which it was observed.

    Analysis¶

    An Analysis is the output of analysis of an observable. It has

    • zero or more observables.
    • free form analysis data.

    Analysis Module¶

    An Analysis Module is what is responsible for taking an observable and generating analysis. Analysis modules are loaded and executed by Engines.
